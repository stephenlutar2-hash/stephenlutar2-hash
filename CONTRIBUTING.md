# Contributing

Thanks for your interest in contributing.

This repository is a **personal profile**, not a multi-contributor project.
For substantive work please head to the relevant repo under
[**SZL Holdings**](https://github.com/szl-holdings):

- [`ouroboros`](https://github.com/szl-holdings/ouroboros) — bounded-loop runtime
- [`ouroboros-thesis`](https://github.com/szl-holdings/ouroboros-thesis) — research papers
- [`szl-holdings-platform`](https://github.com/szl-holdings/szl-holdings-platform) — sovereign engine
- [`.github`](https://github.com/szl-holdings/.github) — org-wide policy, templates, reusable workflows

## How to engage

| Type of contribution | Where to go |
|---|---|
| Bug report on a product | Open an issue on that product's repo |
| Feature request | Open an issue on the product or platform repo |
| Security vulnerability | See [`SECURITY.md`](./SECURITY.md) — **do not file a public issue** |
| Academic question on the Ouroboros Thesis | [`ouroboros-thesis` Discussions](https://github.com/szl-holdings/ouroboros-thesis/discussions) |
| Press / partnerships / hiring | `inquiries@szlholdings.com` |

## Style for any PR you do open here

- One concern per PR
- Conventional Commits (`feat:`, `fix:`, `chore:`, `docs:`, `ci:`, `refactor:`)
- Signed commits where possible
- PRs must pass any CI configured on the target repo

All interactions are governed by the [Code of Conduct](./CODE_OF_CONDUCT.md).
